# 🎙️ VoiceNotes AI

VoiceNotes AI is a Django-based web application that converts **Voice, Audio and Video into Text Notes**.

Features
- 🎤 Speech to Text (Mic recording)
- 🎬 Video to Text transcription
- 📄 Save notes to database
- 📥 Download notes as TXT
- 🔐 User Login system
- 💻 Beautiful modern UI

Tech Stack
- Python
- Django
- SQLite
- HTML/CSS/JavaScript
- SpeechRecognition
How to Run

1. Clone the repo
git clone https://github.com/R-nithyabharathi/VoiceNotes-AI.git

2. Go to project folder
cd voicenotes

3. Install dependencies
pip install django SpeechRecognition moviepy pydub

4. Run migrations
python manage.py migrate

5. Start server
python manage.py runserver

Open http://127.0.0.1:8000/
